# Bitsight CSV 報表工具

一個 **網頁工具**，可上傳、處理 Bitsight CSV 資料，並生成互動式報表。  
使用者可以選擇要顯示的欄位、套用風險向量，並將表格以 **Word 友善格式** 複製。

---

## 功能特色

- 上傳 CSV 檔案
- 選擇報表要顯示的欄位（表頭）
- 選擇 Risk Vector（風險向量）
- 在頁面即時預覽 CSV 資料
- 支援拖放排序已選表頭
- 生成 Word 友善表格
- 可直接複製表格到剪貼簿，方便貼入 Microsoft Word
- 使用 `localStorage` 暫存 CSV 資料與表頭設定
